A1 · Стек AI-студии — май 2026

Матрица «модальность × инструмент × кейс». Где агенты пишут код, где дирижируют сервисами. Точка опоры для всего Advanced-трека.

Что строим

Карта стека, на котором реально работает AI-студия в мае 2026. Не «список любимых инструментов», а decision-table: для какой задачи какой движок, какой средний чек на единицу выхода, какие ограничения. Карта — точка опоры для оставшихся 19 глав Advanced.

Две роли агента в студии

В студии 2026 агент работает в двух режимах. Их нельзя путать — отличаются инструментарий, метрики и риски.

РольЧто делаетАртефактГде сложность
Code agentПишет код, чинит баги, скаффолдит проектDiff, PR, бинарникКорректность + ревью
Pipeline orchestratorДирижирует сервисами генерации (image/video/audio/CV)Файл/кадр/секвенция/инсталляцияСтоимость, latency, идемпотентность

Code agent — то, что вы освоили в Beginner и Middle (Claude Code, Codex, Cursor). Pipeline orchestrator — это то, чем Advanced занимается отдельно: вы пишете не код, а программу, которая зовёт FLUX, Seedance, Higgsfield, RunPod, TouchDesigner.

В реальной студии обе роли стоят рядом: code agent пишет orchestrator, который вызывает генеративные сервисы.

Когда какой инструмент

Не бывает «универсального лучшего». Решение по трём осям:

  1. Стоимость единицы выхода — image и video на масштабе дороже LLM на порядки.
  2. Latency — стенд требует <1 c, креатив для СММ — терпит минуты.
  3. Privacy — приватный архив и клиентские данные не уходят в облако.

Реальный кейс: бренд-агентство WEDOIT

Стек одного боевого проекта, разнесённый по ролям:

  • Бренд-токены (палитра #0A0A0A + #C8FF00, Inter) — машинно-читаемый JSON, MDM design-system в Figma. Передаются в промпт-шаблоны.
  • Код сайта/КП — Claude Code пишет Astro + Tailwind на токенах, ревьювит Cursor.
  • Креатив для соцсетей — FLUX 2 для статики, Seedance 2.0 для motion, Higgsfield Soul Cinema если нужен ведущий с лицом.
  • Клиент-ревью — статический сайт (flag-review шаблон) + Cloudflare Worker + KV под лайки/комментарии. Каждое ревью клонируется за 30 минут.

Code agent работает с первым, вторым и четвёртым пунктом. Pipeline orchestrator — с третьим. Один проект — обе роли.

Подводные камни

  • «У нас же есть GPT, он всё умеет» — нет. На массовых задачах разница в стоимости и качестве между специализированными движками и универсальными LLM измеряется десятками раз.
  • «Локалка дешевле» — только если у вас есть свободное GPU-время и задача не требует свежей модели. На больших объёмах cloud Batch API часто выходит дешевле собственного железа с учётом амортизации.
  • «Подберём инструмент в процессе» — нельзя. Карту стека для проекта фиксируем до начала, иначе теряем бюджет на сравнение в проде.
  • Версии движков меняются раз в 1–3 месяца — карту пересматриваем перед каждым новым проектом. Возможные источники: arena leaderboards (image, video, code), changelogs провайдеров.

Recap

  • Агент в студии — это или code agent, или pipeline orchestrator. Инструменты и метрики разные.
  • Карта стека = таблица «модальность × инструмент × стоимость × кейс». Без неё бюджет не контролируется.
  • Решение по трём осям: стоимость единицы, latency, privacy.
  • Карту пересматриваем перед каждым проектом; цены и версии меняются раз в 1–3 месяца.
