A1 · Стек AI-студии — май 2026

Матрица «модальность × инструмент × кейс». Где агенты пишут код, где дирижируют сервисами. Точка опоры для всего Advanced-трека.

Что строим

Карта стека, на котором реально работает AI-студия в мае 2026. Не «список любимых инструментов», а decision-table: для какой задачи какой движок, какой средний чек на единицу выхода, какие ограничения. Карта — точка опоры для оставшихся 19 глав Advanced.

Две роли агента в студии

В студии 2026 агент работает в двух режимах. Их нельзя путать — отличаются инструментарий, метрики и риски.

РольЧто делаетАртефактГде сложность
Code agentПишет код, чинит баги, скаффолдит проектDiff, PR, бинарникКорректность + ревью
Pipeline orchestratorДирижирует сервисами генерации (image/video/audio/CV)Файл/кадр/секвенция/инсталляцияСтоимость, latency, идемпотентность

Code agent — то, что вы освоили в Beginner и Middle (Claude Code, Codex, Cursor). Pipeline orchestrator — это то, чем Advanced занимается отдельно: вы пишете не код, а программу, которая зовёт FLUX, Seedance, Higgsfield, RunPod, TouchDesigner.

В реальной студии обе роли стоят рядом: code agent пишет orchestrator, который вызывает генеративные сервисы.

Когда какой инструмент

Не бывает «универсального лучшего». Решение по трём осям:

  1. Стоимость единицы выхода — image и video на масштабе дороже LLM на порядки.
  2. Latency — стенд требует <1 c, креатив для СММ — терпит минуты.
  3. Privacy — приватный архив и клиентские данные не уходят в облако.

Реальные кейсы из индустрии

Чтобы карта стека не оставалась абстракцией — четыре публично задокументированные работы эталонных студий генеративного искусства и медиа-инсталляций.

Refik Anadol Studio — «Unsupervised — Machine Hallucinations: MoMA» (2022–2023)

Большая аудио-визуальная инсталляция в Музее современного искусства Нью-Йорка. Студия Анадола обучила StyleGAN-семейство моделей на датасете коллекции MoMA (около 138 000 произведений) и развернула real-time inference на LED-стене ~7 × 7 метров. Аудио синхронизировано с визуалом через OSC. Полгода непрерывной экспозиции с автоматическим перезапуском воркеров.

Урок: на масштабе долгоидущей экспозиции обязательно прорабатывается uptime — система должна перезапускаться сама без оператора. Это тема модулей A11–A12: telemetry → control loop, heartbeat.

TeamLab — Borderless (Tokyo / Shanghai / Jeddah)

Японский коллектив TeamLab держит постоянные экспозиции в нескольких городах. Внутри — сетка из сотен GPU-станций, синхронизированных в единую сцену через TouchDesigner и Spout. AI-компонент: CV-распознавание движения посетителей плюс генеративные визуалы, реагирующие на поведение аудитории.

Урок: это distributed real-time система. Одной мощной машины недостаточно — нужна сеть согласованных нод. Модули A9 (TD + StreamDiffusion) и A12 (telemetry) разбирают этот класс архитектуры детально.

OUCHHH — «Poetic AI» в Atelier des Lumières (Париж)

Турецкая студия проецировала AI-сгенерированный визуал, обученный на корпусе текстов Малларме, Бодлера и Рембо, на стены и пол парижского пространства Atelier des Lumières. Площадка использует порядка 140 проекторов с overlap-зонами, и медиа-сервер выравнивает кадры. ML-модель отдаёт поток в формате, совместимом с media-server.

Урок: физический слой формирует limits ML-стека. Сто сорок проекторов требуют отдельной системы калибровки и выравнивания. Модель не может «генерить как хочется» — формат картинки диктует площадка.

Random International — «Rain Room» (Barbican / MoMA / LACMA / Sharjah)

Британская студия Random International. Перформативная инсталляция, в которой дождь физически прекращается над посетителем. Технический слой: depth-камеры отслеживают людей, control-система отключает соответствующие секции water-grid в потолке.

Урок: AI в инсталляции — это не обязательно генеративка. Здесь главное — CV-слой (трекинг людей) и control loop с физическим эффектором (соленоидами, регулирующими подачу воды). Класс работ, в которых LLM появляется только на подготовительной стадии — для документации, заявки, technical rider. Тема модулей A11 (interactive booths) и A12.

Общий скелет

Все четыре кейса используют одинаковую архитектуру:

  1. Pipeline orchestrator (часто TouchDesigner для real-time, Python для подготовки данных).
  2. AI-модель под задачу — генеративная (Anadol, OUCHHH) или дискриминативная (Random, TeamLab CV).
  3. Физический слой — проекторы / LED, сенсоры (RGB, depth), аудио-выход, опционально DMX-свет и physical actuators.
  4. Control loop с heartbeat-мониторингом и автоматическим перезапуском.

Это минимальная архитектура инсталляции класса «постоянная экспозиция» 2024–2026.

На своём масштабе: четыре профиля

У эталонных студий стек получается похожим, но профили нагрузки различаются. Тот же скелет масштабируется вниз так:

Креативное агентство на 5–8 человек

Состав: 2 креативных директора, 2 дизайнера, 1 motion, 1 техно-художник, 1 продюсер. Делают immersive-кампании для брендов — projection mapping на product launch, иммерсивные стенды на выставках, AR-фильтры.

Стек: FLUX 2 + Nano Banana 2 (статика), Seedance 2.0 + Kling 3.0 (motion offline), TouchDesigner + StreamDiffusion (real-time для стендов), Soul ID (повторяющиеся персонажи), Sonnet 4.6 + Opus 4.7 (текст).

Месячно: ~$400–700 на генеративку. Капекс на железо: $15–30k (короткофокусный проектор 5000–7000 лм типа Epson EB-PU2200B, RealSense D455, рабочая станция с RTX 4090).

Solo media-artist, готовящий заявки на open calls

Один человек с авторской практикой. Готовит заявки на Ars Electronica, transmediale, NODE Forum, MUTEK, Lumen Prize. Месячный объём — одно-два полноценных portfolio-видео плюс рендер-документация.

Стек: FLUX 2 + GPT Image 2 (визуалы для documentation и application form), Seedance 2.0 + Higgsfield Soul Cinema (cinematic teaser), TouchDesigner + StreamDiffusion (прототип инсталляции на 4070), Opus 4.7 (artist statement — важна языковая чистота), FLUX LoRA Fast (consistency между кадрами).

Месячно: ~$80–150 на генеративку. Капекс: $5–10k (BenQ TH685 или аналогичный 3000 лм, Kinect Azure или RealSense, станция с RTX 4070/4080).

AV-студия / VJ-collective на 2–5 человек

Делают live visuals для events (концерты, festivals, club nights). Tour-ready setup.

Стек: TouchDesigner как ядро, StreamDiffusion + FLUX schnell (AI-augmented visuals), Resolume Arena (классическая VJ-практика), FLUX 2 (подготовка сцен), Runway Gen-4 Unlimited (loop-видео в сэты), Soul ID (повторяющийся артист), Ableton Live + аудио-реактивные CHOPs в TD.

Месячно: $200–500 на генеративку (большая часть real-time идёт на собственном железе). Капекс: $20–40k (1–3 проектора 8000+ лм для concert stage, MIDI-контроллер, RME Fireface UCX II или Focusrite Scarlett 18i20, ноутбук с RTX 4090).

Резидент-инженер при галерее или музее

Один штатный технический специалист. Задачи: поддержка постоянных инсталляций, монтаж/демонтаж временных выставок, technical rider для приглашённых артистов, документация.

Стек: TouchDesigner как универсальный «клей», MediaPipe + depth-камеры (touchpoint’ы и трекинг посетителей), Sonnet 4.6 (техническая документация и maintenance log), Claude Code (watchdog’и, скрипты автозапуска, утилиты для калибровки), LoRA-тренировка стиля института.

Месячно: $200–400 на генеративку. Капекс: бюджет институции — проекторы 10 000+ лм класса Christie / Panasonic / Barco, mediaserver-машины (Modulo Player, Watchout), сеть DMX/Art-Net контроллеров (MA Lighting, ETC), multichannel аудио (RME MADIface XT).

Физический слой: проекция, сенсоры, аудио, свет

В отличие от чисто-software AI-продуктов, студии в инсталляционном домене обязаны держать в стеке физический слой. Эта часть не масштабируется как облачные API — нужны капитальные вложения, складское хранение, страховка и логистика.

Аудио-интерфейсы

Для многоканального звука в инсталляциях — RME Fireface UCX II (8 in / 8 out), Focusrite Clarett+ 8Pre, или MADI-системы (RME MADIface XT) в распределённых установках. Для VJ — стерео-входа Focusrite Scarlett Solo хватает: забирается main signal с мастер-bus’а пульта.

Управление светом — DMX-512 и Art-Net

Свет в инсталляциях управляется через индустриальные протоколы:

  • DMX-512 — классика, 5-pin XLR, до 512 каналов на универс.
  • Art-Net / sACN — сетевые протоколы поверх Ethernet, переносят несколько универсов. Поддерживаются TouchDesigner из коробки и большинством mediaserver’ов.

Контроллеры — консоли MA Lighting / ETC; software — Lightkey, Q LAB. Световые приборы — Robe, Chauvet Professional, ETC Source Four. Для LED-стен отдельный класс (Absen, ROE Visual).

Что от вас ждут на open call

Минимальный application pack для AI/installation работы:

  1. Artist statement (300–500 слов) — концепция работы, контекст в практике автора.
  2. Project description (500–1000 слов) — техническое описание, что демонстрирует работа, риски.
  3. Documentation video (2–5 минут) — монтаж демо + интервью + behind-the-scenes.
  4. Technical rider — список оборудования (своё / запрашиваемое), пространственные требования, мощность, кабельная инфраструктура.
  5. Budget breakdown — на что пойдут production-средства если их выделяют.
  6. CV и портфолио — ссылки на предыдущие работы.

Claude Opus 4.7 хорошо подходит для шлифовки artist statement — важна языковая точность. Но черновик пишет автор сам: кураторы считывают AI-сгенерированные тексты на глаз и относятся прохладно. ИИ полезен в подготовке documentation video (см. A5, A7, A8) и при составлении technical rider — Claude Code умеет сгенерировать аккуратный markdown по шаблону.

Подводные камни

  • «У нас же есть GPT, он всё умеет» — нет. На массовых задачах разница в стоимости и качестве между специализированными движками и универсальными LLM измеряется десятками раз.
  • «Локалка дешевле» — только если у вас есть свободное GPU-время и задача не требует свежей модели. На больших объёмах cloud Batch API часто выходит дешевле собственного железа с учётом амортизации.
  • «Подберём инструмент в процессе» — нельзя. Карту стека для проекта фиксируем до начала, иначе теряем бюджет на сравнение в проде.
  • «Купим оборудование под конкретный проект» — capex на проекторы и сенсоры окупается только при многократном использовании. Под single-shot проект оборудование берут в аренду у профессиональных rental-компаний (AV Stumpfl, PRG, Eclipse Staging Services).
  • «AI-сгенерированный artist statement сэкономит время» — кураторы фестивалей считывают за минуту. Финальный текст — авторский; AI допустим как редактор стилистики.
  • Копирование стека крупного игрока 1-в-1 — стек Refik Anadol Studio имеет смысл при её объёме production; для solo media-artist это оверинжиниринг и кассовый разрыв.
  • Версии движков меняются раз в 1–3 месяца — карту пересматриваем перед каждым новым проектом. Источники: arena leaderboards, changelogs провайдеров.

Recap

  • Агент в студии — это или code agent, или pipeline orchestrator; в инсталляционной практике добавляется третья роль — оператор площадки.
  • Карта стека = таблица «модальность × инструмент × стоимость × кейс» плюс физический слой (проекция / сенсоры / аудио / свет). Без неё бюджет не контролируется.
  • Решение по трём осям: стоимость единицы, latency, privacy.
  • Эталонные студии (Refik Anadol Studio, TeamLab, OUCHHH, Random International) и малые команды используют один скелет — различаются профили нагрузки, capex и доля code-agent деятельности.
  • Open calls (Prix Ars Electronica, transmediale, MUTEK, NODE Forum, Lumen Prize, резидентства) — основной канал входа в индустрию.
  • Карту пересматриваем перед каждым проектом; цены и версии меняются раз в 1–3 месяца.
