A6 · Marketplace + product photoshoot

Карточки для маркетплейсов (Wildberries/Ozon-style) и бренд-фото товара: lifestyle, closeup, CGI, restyle. Чтобы выход проходил модерацию и продавал.

Что строим

Два набора артефактов под коммерческие задачи:

  1. Marketplace cards — главное фото (compliant с правилами площадки), 5–7 secondary, A+ контент-модули.
  2. Product photoshoot — бренд-фото товара в режимах: product_shot, lifestyle_scene, closeup_product_with_person, moodboard_pin, hero_banner, social_carousel, ad_creative_pack, virtual_model_tryout, conceptual_product, restyle.

Разница: marketplace заточен под compliance модерации (белый фон, отсутствие надписей, цвет товара = реальному), photoshoot — под бренд и виральность.

Marketplace pipeline

ВХОД                   ОБРАБОТКА                    ВЫХОД
─────────────────────────────────────────────────────────────
исходное фото товара  →  фон → белый                 main.jpg
описание SKU          →  цвет нормализация           main_white_bg.jpg
требования площадки   →  пропорции 3:4               secondary_lifestyle.jpg
                         генерация secondary         secondary_closeup.jpg
                         A+ модули                   aplus_module_1..N.jpg
                         compliance проверка         report.json

Compliance-проверка обязательна перед загрузкой: модератор отклоняет за надпись на main, неправильный bg, обрезанный товар, лишние товары в кадре. Чек-лист поставляется как promp + post-validator.

Skill для маркетплейса

В репо подключён skill higgsfield-marketplace-cards. Он скрывает compliance-референсы за CLI: на входе — описание товара и категория, на выходе — комплект карточек.

/higgsfield-marketplace-cards \
    product="зимние ботинки женские, чёрная кожа, шерсть" \
    category="обувь" \
    sku="BT-2026-W-39" \
    refs=./input/boots_studio.jpg

Что важно: skill сам подбирает шаблоны промптов под платформу (WB/Ozon-style). Не пишите промпты руками — теряете compliance.

Реальный кейс: серия для е-com бренда

Задача: 1 SKU, 7 артефактов (main + 4 secondary + 2 A+).

Stack:

  • main + 1 secondary closeup — product_shot + closeup_product_with_person через Higgsfield;
  • 3 secondary lifestyle — lifestyle_scene (городской вечер, дома, спорт);
  • 2 A+ модуля — GPT Image 2 (характеристики + сравнение с конкурентом).

Стоимость одного SKU: ~$3 (7 × $0.05–0.07 + overhead). Время: 20–30 минут с проверкой compliance. До генеративки этот объём = студийная съёмка за 1–2 дня + ретушёр, бюджет от $300.

Compliance post-validator

def check_main(image_path: Path) -> list[str]:
    img = cv2.imread(str(image_path))
    errors = []
    if not is_white_bg(img, threshold=240):
        errors.append("main: bg не белый (avg < 240)")
    if has_text_overlay(img):
        errors.append("main: обнаружен текст на изображении")
    if has_multiple_products(img):
        errors.append("main: подозрение на несколько товаров в кадре")
    aspect = img.shape[1] / img.shape[0]
    if not (0.7 <= aspect <= 0.8):
        errors.append(f"main: пропорции {aspect:.2f} вне 3:4")
    return errors

Локальная проверка дешевле, чем reject модератором и переделка. Прикручивается перед заливкой.

Подводные камни

  • «Сгенерил красивое — оно зашло» — main без проверки compliance обычно отклоняется. Visual quality ≠ маркетплейс quality.
  • Цвет товара отъезжает на 10–15% — критично для одежды и косметики. Color normalization + контрольный кадр с цветовой картой.
  • Virtual try-on на низкобюджетной модели — выглядит unhinged: лица плывут, пропорции некорректны. Используйте Soul ID или загружайте reference модели.
  • A+ модули перегружают информацией — пользователь смотрит секунду. 1 тезис на модуль, не 5.
  • ip_detected у Higgsfield — ловится ДО создания, blur помогает частично (см. memory reference_higgsfield_cli). Используйте обезличенные фото-референсы.

Recap

  • Marketplace и photoshoot — разные пайплайны. Первый — про compliance, второй — про бренд.
  • Не пишите промпты руками: skills (higgsfield-marketplace-cards, higgsfield-product-photoshoot) хранят правила сборки.
  • 7 карточек ≈ $3 и 30 минут; vs студия — порядок в 100 раз дешевле.
  • Compliance post-validator — обязательный шаг перед заливкой.
