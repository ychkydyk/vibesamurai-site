A5 · Image generation для бренда

FLUX 2, GPT Image 2, Nano Banana 2/Pro — когда какой движок. Промпт-шаблоны на бренд-токенах, tokenomics gate, реальный кейс для агентства.

Что строим

Production-пайплайн для генерации image-креатива под бренд: hero для landing, серия для СММ, обложки для статей. Главное — повторяемое качество в фирменном стиле, не «удачный кадр раз в десять попыток».

Не пытайтесь побороть слабую сторону движка промптом — переключайтесь.

Промпт-шаблон на бренд-токенах

Из A4 — бренд-токены содержат image_prompt_template. Финальный промпт собирается из трёх слоёв:

[brand prefix] + [scene] + [negative suffix]
   ↑                ↑           ↑
   фиксирован       варьируется фиксирован
   (≤30 слов)       (≤50 слов)  (≤20 слов)

Пример:

PREFIX:  high-contrast editorial, near-black background #0A0A0A,
         single chartreuse accent #C8FF00, Inter sans-serif typography,
         minimal composition, --ar 4:5
SCENE:   two figures from above, working at a desk, no faces visible,
         laptop closed, paper notes
NEGATIVE: neon glow, rainbow, vintage film grain, cute mascot

Если бренд-prefix плывёт между постами — серия теряет консистентность. Фиксируйте prefix как часть design system, не как часть промпта.

Tokenomics gate (обязательно)

Перед любой партией image-генераций — расчёт стоимости и явное подтверждение от автора. Это правило feedback_tokenomics_confirm и feedback_analysis_tokenomics_gate.

def estimate(engine: str, count: int) -> dict:
    cost_per = {"flux-2": 0.04, "gpt-image-2": 0.07, "nano-banana-2": 0.05}
    return {
        "engine": engine,
        "count": count,
        "total_usd": cost_per[engine] * count,
        "approx_minutes": count * 8 / 60,
    }

est = estimate("flux-2", 200)
print(f"Партия {est['count']} кадров через {est['engine']}: "
      f"${est['total_usd']:.2f}, ~{est['approx_minutes']:.1f} мин.")
confirm = input("[y/N] продолжаем? ").strip().lower()
if confirm != "y":
    raise SystemExit("отменено")

200 кадров × $0.04 = $8. Без gate это уходит незаметно; с gate — осознанное решение.

Реальный кейс: серия из 12 постов для агентства

Постановка: 12 еженедельных постов под бренд (#0A0A0A + #C8FF00 + Inter). Каждый пост — hero-кадр + 6 секунд motion-pin (motion будем делать в A7).

Стек:

  • бренд-токены brand-tokens.json (см. A4);
  • posts/q2.yaml — 12 записей (post_id, scene, mood);
  • runner — FAL Python SDK поверх FLUX 2;
  • кэш — disk + JSON-сайдкары (см. A2);
  • ffmpeg + Pillow — для overlay лого и subtitle.
import fal_client
from pathlib import Path

prefix = brand["image_prompt_template"]
negs   = ", ".join(brand["negatives"])

for post in posts:
    seed = stable_seed(post["id"])
    prompt = f"{prefix} — {post['scene']}"
    out, meta = run_step("image:hero", {
        "engine": "flux-2",
        "prompt": prompt,
        "negative": negs,
        "seed": seed,
        "ar": "4:5",
    }, runner=fal_flux_runner)
    Path(f"posts/{post['id']}_hero.png").write_bytes(out.read_bytes())

Результат: за один прогон — 12 кадров, $0.48, ~2 минуты. Любой пост можно перегенерить с новым scene, остальные 11 берутся из кэша.

Подводные камни

  • Промпт без --ar — движок отдаёт по умолчанию 1:1, а у вас лента 4:5. Aspect ratio — часть payload, не часть scene.
  • Цвета #0A0A0A в промпте — FLUX 2 их понимает, но не всегда точно попадает. Закрепляйте через 1–2 рефренс-картинки или fixed seed-per-brand.
  • Текст на FLUX — не пытайтесь. Текстовые элементы — на GPT Image 2 или поверх в Figma/Photoshop пост-обработкой.
  • Multi-ref в одном вызове — Higgsfield CLI auto-coerce-ит multi-ref в start_image (см. memory reference_higgsfield_cli). Сводите задачу к одному ref или используйте Nano Banana 2 с явным reference-полем.
  • Auto-upload падает — у Higgsfield CLI бывают сбои auto-upload. Загружайте файлы заранее, передавайте upload_id.

Recap

  • Три движка для разных задач: FLUX 2 (фотореал, editorial), GPT Image 2 (текст, дизайн), Nano Banana 2 (references, character).
  • Промпт = brand prefix (фиксирован) + scene (вариативно) + negative suffix (фиксирован).
  • Tokenomics gate перед партией — обязательный шаг, не опция.
  • Кэш + lineage (из A2) делают итерации дёшевыми: меняем одну scene — пересчитываем один кадр.
