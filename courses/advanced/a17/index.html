A17 · Token economics через модальности

Стоимость кадра, секунды видео, 1k токенов. Каскад Haiku→Sonnet→Opus + prompt caching + Batch API. Обязательный gate перед платной генерацией.

Что строим

Единая система учёта стоимости через все модальности студии: text (LLM), image, video, audio. Цель — заранее знать, сколько стоит проект, видеть драйверы стоимости, обязательно подтверждать перед платной генерацией.

Каскад Haiku → Sonnet → Opus

Из practiked (memory feedback_course_translation_pipeline): для пакетных задач (типа перевод курса) используем каскад: Haiku → Sonnet → Opus + prompt caching + Batch API. Примеры в чат — Sonnet, финал — Opus.

Логика:

ВСЕ ЗАПРОСЫ ──► Haiku ($0.8/1M)         ──► 80% задач закрыты дёшево
                  │
       neсложно?  │
                  ├── да ──► OUTPUT
                  │
                  └── нет ──► Sonnet ($3/1M)     ──► 18% задач
                              │
                              ├── приемлемо? ──► OUTPUT
                              │
                              └── нет ──► Opus ($15/1M)  ──► 2% задач
                                            │
                                            └──► OUTPUT

Эффективная стоимость на запрос: 0.8 * 0.8 + 3 * 0.18 + 15 * 0.02 = 0.64 + 0.54 + 0.30 ≈ $1.48 / 1M. Это в 10 раз дешевле, чем гонять всё через Opus.

Prompt caching

При повторных вызовах с одинаковым префиксом — Anthropic кэширует префикс на 5 минут. Стоимость cached input → 10% от обычной.

# первый вызов: кэширует префикс
client.messages.create(
    model="claude-sonnet-4-6",
    messages=[{"role": "user", "content": [
        {"type": "text", "text": LONG_SYSTEM_PROMPT, "cache_control": {"type": "ephemeral"}},
        {"type": "text", "text": USER_QUESTION},
    ]}],
)

# второй и далее (в течение 5 минут): префикс из кэша, -90% input cost

Когда оправдано: партии запросов с одинаковым system prompt; интерактивный агент с длинным CLAUDE.md. Прямой ROI: на стенде с 1000 вопросов/час и system prompt в 8k токенов prompt caching экономит десятки долларов в день.

Batch API

Anthropic Batch API: запросы выполняются в течение 24 часов (но обычно сильно быстрее), стоимость — 50% от обычной.

Когда оправдано:

  • не интерактивные задачи (ночной курсовой ingest, перевод, классификация архива);
  • партия от 1000 запросов;
  • ok ждать минуты-часы.

В каскаде Haiku → Sonnet → Opus прогон Batch API даёт ещё −50%. Combined с prompt caching — до −80% против «голого» интерактива.

Tokenomics gate (обязательно)

Главное правило студии (memory feedback_tokenomics_confirm + feedback_analysis_tokenomics_gate): перед любой платной генерацией (LLM-анализ, image, video) — считаем стоимость и ждём явного [y/N] подтверждения.

Без бэк-доры, без bypass-флагов. И в админке, и в CLI.

def gate(label: str, cost_usd: float):
    print(f"\n=== TOKENOMICS GATE ===")
    print(f"Задача: {label}")
    print(f"Оценка: ${cost_usd:.2f}")
    confirm = input("[y/N] продолжить? ").strip().lower()
    if confirm != "y":
        raise SystemExit("отменено пользователем")

Это не «осторожность для слабонервных». Один забытый цикл по 200 видео — $480 за пять минут. Gate ловит это до того, как деньги уходят.

Реальный кейс: ночной перевод курса

Задача: перевести Beginner + Middle (32 главы × ~10k символов = 320k символов исходника). Каскад:

  1. Haiku — первичный перевод всех 32 глав (max_tokens=12000). Стоимость: $32 × 12000 / 1M × 0.8 ≈ $0.31.
  2. Sonnet — review результата Haiku, отметка глав, требующих доводки. Стоимость: $0.6.
  3. Opus — финал по отмеченным главам (5 из 32). Стоимость: $5 × 12k × 15 / 1M = $0.90.

Итог: ~$2 за полный перевод курса (Batch API + prompt caching стянут в ~$1).

В лоб через Opus: 32 × 12k × 15 / 1M = $5.76. Разница не катастрофическая на 32 главах, но при масштабе (300 материалов) — десятки долларов экономии.

Подводные камни

  • Считать input-токены тоже — output дороже, но input на больших prompts тоже копит. Особенно с RAG (большой контекст).
  • Caching работает только при идентичном префиксе — измените 1 символ в system prompt → кэш промахнулся.
  • Batch API не для интерактива — ответы приходят пачкой, не in real-time.
  • «Локалка бесплатно» — нет. 4070 на 24/7 inference = +200–300W = реальные деньги на электричестве.
  • Скрытые расходы — Soul ID training, S3 storage для кэша, Cloudflare bandwidth. Включайте в учёт.

Recap

  • Цены меняются раз в 1–3 месяца — пересверяемся перед каждым проектом.
  • Один image ≈ 500 LLM-вызовов; один видео-клип ≈ 1800 LLM-вызовов. Это меняет архитектуру.
  • Каскад Haiku → Sonnet → Opus экономит ~10× против «всё через Opus».
  • Prompt caching + Batch API — −80% на партиях.
  • Tokenomics gate перед платной генерацией — обязателен. Без bypass.
